## Overview

## HELLOs  this is for testing on mintlify editing 

This site uses [Mintlify](https://mintlify.com). Add your own pages as `.mdx` files and register them in `docs.json` under `navigation`.

## Next steps

1. Edit this page or add new `.mdx` files in the repository root (or in subfolders—reference paths without the extension in `docs.json`).
2. Update `docs.json` so new pages appear in the sidebar.
3. Run the Mintlify CLI locally: `mint dev` (install with `npm i -g mint`).
4. Push changes to GitHub; Mintlify can deploy from your connected repo.
